#  Fundamentos De React

Este proyecto fue desarrollado como parte de la materia de **Desarrollo Web**, donde se aplican los conceptos principales mediante ejemplos prácticos e interactivos.

---

## Tecnologías utilizadas

* React
* JavaScript
* Tailwind CSS
* Zustand
* LocalStorage

---

## Contenidos implementados

En este proyecto se trabajaron los siguientes temas:

### useState

Manejo de estados dentro de los componentes para actualizar la interfaz dinámicamente.

### useEffect

Ejecución de efectos secundarios como consumo de APIs y carga de datos.

### Custom Hooks

Creación de lógica reutilizable para simplificar el código.

### Props

Comunicación entre componentes padre e hijo.

### Zustand

Manejo de estado global sin necesidad de props.

### Renderizado condicional y listas

Mostrar contenido dinámico dependiendo de condiciones y recorrer arreglos con `map`.

### LocalStorage

Almacenamiento de datos en el navegador para mantener información persistente.

## Cómo ejecutar el proyecto

1. Clonar el repositorio:

```bash
git clone https://github.com/TU-USUARIO/TU-REPOSITORIO.git
```

2. Instalar dependencias:

```bash
npm install
```

3. Ejecutar el proyecto:

```bash
npm run dev
```
